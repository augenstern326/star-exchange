## 🆘 快速求助

| 问题 | 解决方案 |
|------|---------|
| "DATABASE_URL is not set" | 检查 `.env.local` |
| "relation 'users' does not exist" | 访问 `/api/init` 初始化 |
| "permission denied" | 检查 Neon 用户权限 |
| "password verification failed" | 检查用户名和密码 |
| 数据库连接慢 | 检查网络，查看 Neon Console |
| 登录失败 | 查看浏览器控制台错误 |

详见各文档的 FAQ 和故障排除部分。
